
# PurePods Web

This repository contains the working files for the **PurePods website redesign**.

It is used as a shared workspace to:
- structure content
- organise images
- define copy and experience guidelines
- prepare the future production website

## Tech context
The final website will be built with **Next.js + Sanity**, but this repository is not the production codebase.




## Status
Work in progress.  
Content, images and structure may change.
